# Photoreal Hero Background (Bitmap strokes)

This version uses real photographed foundation brush-stroke images with **alpha transparency**:
- `public/stroke_1.webp`
- `public/stroke_2.webp`
- `public/stroke_3.webp`

No procedural blob painting is used for the hero background.

If you want to swap to different strokes, replace those files with any high-res RGBA/transparent WebP/PNG strokes and keep the same filenames.
